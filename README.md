# System Analysis Project – Sistem Persewaan

## Deskripsi
Repository ini berisi project analisis sistem informasi
untuk sistem persewaan yang dikembangkan berdasarkan
studi kasus pada mata kuliah Metodologi Penelitian

## Tujuan Project
Melakukan analisis kebutuhan sistem dan pemodelan sistem
sebagai dasar pengembangan sistem informasi persewaan.

## Ruang Lingkup
- Analisis proses bisnis sistem persewaan
- Identifikasi kebutuhan fungsional dan non-fungsional

## Output
- Dokumentasi proses bisnis
- Dokumen kebutuhan sistem

## Peran
System Analyst

## Catatan
Project ini bertujuan sebagai dokumentasi pembelajaran
dan analisis sistem informasi.
